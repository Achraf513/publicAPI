PUBLIC API 1.0.0

[ Base URL: publicapi51.herokuapp.com/ ]

This API provides endpoints to get random customers and merchants data.

MODELS

CUSTOMER MODEL

{

"_id": int,
"email": string,
"username": string,
"picURL": string(URL),
"password": string,
"phone": string,
"address": string,
"balance": int,

}

PRODUCTS MODEL

{

"_id": int,
"title": string,
"description": string,
"picURL": string(URL),
"category": string,
"rating": float,
"stock": int,
"price": int,

}

How to use it

Public API can be used with any type of shopping project that needs products and customers in JSON format. you can use examples below to check how PublicAPI works and feel free to enjoy it in your awesome projects!

Customers

Get All Customers

In this case the server will return all the customers saved in it's data

fetch('https://publicapi51.herokuapp.com/customers')
.then(res=>res.json())
.then(json=>console.log(json))
